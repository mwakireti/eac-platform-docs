# EAC Platform Images

This folder contains placeholder images for the documentation site.

## How to Add Your Own Images:

1. **Replace these placeholder files** with your actual images:
   - `getting-started.jpg` - Image for the Getting Started section
   - `account-registration.jpg` - Image for Account Registration
   - `user-dashboard.jpg` - Image for User Dashboard
   - `organisation-registration.jpg` - Image for Organisation Registration
   - `teams-collaboration.jpg` - Image for Teams & Collaboration
   - `staff-portal.jpg` - Image for Staff Portal
   - `event-management.jpg` - Image for Event Management
   - `results-scoring.jpg` - Image for Results & Scoring
   - `approvals.jpg` - Image for Approvals
   - `reports.jpg` - Image for Reports
   - `notifications.jpg` - Image for Notifications & Communication
   - `security-privacy.jpg` - Image for Security & Privacy
   - `mobile-access.jpg` - Image for Mobile Access
   - `troubleshooting.jpg` - Image for Troubleshooting & Support
   - `best-practices.jpg` - Image for Best Practices
   - `faqs.jpg` - Image for FAQs

2. **Image Requirements:**
   - Format: JPG, PNG, or WebP
   - Recommended size: 500x300px or larger (will be cropped to fit)
   - File size: Under 2MB for optimal loading

3. **Naming Convention:**
   - Use lowercase with hyphens
   - Match the section ID from the code
